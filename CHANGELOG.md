# Changelog

## [1.2.1] - 2025-12-18

### Fixed / 修复
- Fixed issue where lava cluster multiplier configuration value was not being used by PowerGeneration / 修复岩浆集群倍率配置值未被 PowerGeneration 使用的问题 

## [1.2.0] - 2025-12-18

### Added / 新增
- Infinite Dirt Provider / 无限泥土供应器
- Infinite Milk Provider / 无限牛奶供应器
- Mekanism extended providers / Mekanism 扩展供应器：
  - Infinite Heavy Water Provider / 无限重水供应器
  - Infinite Brine Provider / 无限盐水供应器
  - Infinite Lithium Provider / 无限锂供应器
- Dynamic recipe registration system based on mod loading / 动态配方注册系统，根据 mod 加载情况自动调整

### Changed / 变更
- Migrated all recipes from JSON to Java for dynamic registration / 将所有配方从 JSON 迁移到 Java 进行动态注册
- Optimized recipe registration logic for better mod compatibility / 优化配方注册逻辑，提高 mod 兼容性

## [1.1.0] - 2025-12-16

### Added / 新增
- Added API for registering custom providers / 添加 API，支持注册自定义供应器

## [1.0.1] - 2025-12-16

### Added / 新增
- Added crafting recipes / 补充合成配方

- Initial release / 正式发布
- Infinite Cobblestone Provider / 无限圆石供应器
- Infinite Water Provider / 无限水供应器
- Infinite Lava Provider / 无限岩浆供应器
- Power generation system / 发电系统
- Lava cluster bonus system / 岩浆集群加成系统
- Mod metadata and documentation / 补充 mod 元数据和文档
